Projeto acadêmico

Sistema de Gestão de Pacientes

Aplicação web para apoiar o cadastro, acompanhamento e consulta de informações de pacientes com organização e segurança.

Visão geral

Sistema pensado para simplificar a gestão de registros, tornando o acesso às informações mais rápido e estruturado.

Problema resolvido

Processos manuais podem gerar duplicidade, perda de informações e dificuldade no acompanhamento de históricos.

Solução desenvolvida

Interface responsiva com cadastro, listagem, busca e estrutura para gerenciamento de dados clínicos essenciais.

Tecnologias utilizadas

HTML, CSS, JavaScript, Node.js, Express, PostgreSQL, Git e GitHub.

Funcionalidades

  • Cadastro de pacientes.
  • Busca e filtragem de registros.
  • Visualização de dados principais.
  • Base preparada para histórico e autenticação.

Desafios enfrentados

Organizar informações sensíveis em uma experiência clara, objetiva e preparada para boas práticas de segurança.

Aprendizados obtidos

Prática com CRUD, estrutura de dados, telas administrativas e importância da usabilidade em sistemas de gestão.
