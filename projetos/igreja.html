Projeto acadêmico

Sistema de Gestão para Igreja

Sistema web planejado para centralizar informações administrativas, membros, eventos e departamentos de uma instituição religiosa.

Visão geral

Uma aplicação para apoiar a organização interna de igrejas, reunindo dados importantes em um ambiente simples e acessível.

Problema resolvido

Informações espalhadas em planilhas e anotações dificultavam consultas, atualizações e acompanhamento das atividades.

Solução desenvolvida

Criação de módulos para cadastro, consulta e organização de membros, eventos e áreas administrativas.

Tecnologias utilizadas

HTML, CSS, JavaScript, Node.js, Express, PostgreSQL, Git e GitHub.

Funcionalidades

  • Cadastro e edição de membros.
  • Organização de eventos e departamentos.
  • Consultas rápidas por informações principais.
  • Estrutura preparada para autenticação e relatórios.

Desafios enfrentados

Modelar dados de forma clara e criar uma navegação objetiva para usuários com diferentes níveis de familiaridade digital.

Aprendizados obtidos

Melhor compreensão sobre organização de requisitos, modelagem de banco de dados e construção de fluxos administrativos.
